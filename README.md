# Emma's Portfolio

A portfolio website for Emma to showcase her travel photography and interior design work.

## Features

- **Home/Travel Journal:** Masonry layout of travel photos.
- **Interior Design:** Portfolio of interior design projects.
- **About:** Personal bio and story.
- **Contact:** Contact form and details.

## Tech Stack

- [Astro](https://astro.build)
- [Tailwind CSS](https://tailwindcss.com)

## Commands

All commands are run from the root of the project, from a terminal:

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | Installs dependencies                            |
| `npm run dev`             | Starts local dev server at `localhost:4321`      |
| `npm run build`           | Build your production site to `./dist/`          |
| `npm run preview`         | Preview your build locally, before deploying     |
